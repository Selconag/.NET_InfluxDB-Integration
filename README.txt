In order to make this project please read the documentation on Github Page.
If you already read it, then do as follow:
1)Update your Settings.cs file; Renew bucket, organization, connection string and others.
2)Make sure you builded and ran the image of influxdb on a docker container. Or it won't work.
3)Update connection token in appsettings.json, the token must be dook from influxdb page. (Written in influxdb doc)
4)After you set everything, you are ready to go. Good luck!
Document Reference: https://www.influxdata.com/blog/getting-started-with-c-and-influxdb/#heading1
